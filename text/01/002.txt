# 基本計画 第2話
1. 現地で
  1. エルザが魔法の確認
    - まずエルザに魔法の効力を伝える：シーンで
　女騎士エルザが｜呆然《ぼうぜん》とレクトを見つめる。

<エルザ>「おまえ……、レ、レクト様がなさったんですか……」
<レクト>「はい。
<レクト>　魔法で、ちょっと」

　レクトは、敵騎士とドラゴンの筋力が百分の一になっていることを伝えた。
　体に巡る魔力による身体強化も効かない状態異常も一緒だ。
　生命維持に関わる心臓などの力は正常に保たれているところがこの魔法の気が利いた点だろう。
　魔法を解除するまで弱体化はいつまででも続く。

　エルザは説明を受けて敵騎士のところへ行き、レクトの言が確からしいことを確認した。
　すぐに姫メルカにのし掛かる女騎士を通常配置に戻す。
　メルカは少し｜拗《す》ねた顔をしていたが事情は理解している様子だ。
  2. 姫と話を
    - 姫が驚いて感心する：シーンで
    - 転移の話：地の文でさらっと身の上を記憶がないことなども含めて
　すぐ機嫌を直し、レクトの元へやってきて片膝を突いた。

<メルカ>「深く感謝を捧げます」
<レクト>「あ、いや、それほどでも」
<メルカ>「いいえ。
<メルカ>　レクト様がいらっしゃらなければ今頃どうなっていたか。
<メルカ>　どうか私の感謝の気持ちをお受け取りください」
<レクト>「は、はい。
<レクト>　分かりました、受け取ります。
<レクト>　受け取りますから、どうか立ってください」

　レクトは居心地が悪くていけない。
　女の子で、王女様に｜跪《ひざまず》かれるなんて。
　メルカは花のほころぶような笑みで立ち上がった。

<メルカ>「それにしてもレクト様は高名な魔法使いだったんですね」
<レクト>「それは誤解です。
<レクト>　僕はここに来る途中で魔法使いになったというか……。
<レクト>　その、どうしてここにいるのかも分からない始末で」

　
<メルカ>「レクト様がどうしてここにいるのか、それは私が理由です。
<メルカ>　解読できない魔法書に魔力を通して実行したら、レクト様が召喚されてしまいました。
<メルカ>　その、申し訳ございません！」
<レクト>「ちょっとびっくりしましたが、でも召喚してくださったのなら感謝します。
<レクト>　召喚前に僕は、死んでいたと思うので」
  3. ドラゴンの処置
    - 操精魔法をパージして従仕魔法を使う：シーン
    - 従仕の証 => それに向かって吠えろ
  4. 敵騎士の処置
    - 騎士が持っていた縄で縛って馬にくくる
2. 城に到着
  - 城の描写：地の文
  - 場内の中庭に連れて行った：地の文
3. 王と姫とレクトと宰相とエルザで話し合い
  1. 王国の説明
    - 王の挨拶：シーン
    - 異世界から来たそうだがこの世界はテイルヴィエストと呼ばれている
    - 王国の説明：地の文
  2. 何があったかすでに伝わっている：地の文
  3. レクトの自己紹介
    - およそ姫から王に伝わっている
      - 王が◯◯だそうだな => レクトが補足
    - 元の世界に帰る方法はない
      - レクト的に記憶のない生活に戻る気もないし魔法が使えてうれしかった
  4. 姫が何をしていたか
    - 魔法書は燃えてしまった
  5. 召喚◯◯の話
    - 伝説に伝わる転界者
    - 他国が企んでいるようだ
  6. 姫の熱っぽい視線と王の嫌な顔
  7. レクトの処遇 => 男爵の位、当面は力を調べる、姫預かり（王は嫌そう）
